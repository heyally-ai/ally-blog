New: Ally Web Clipper browser extension for Safari on iOS and macOS

To create a PKM and personal productivity system that actually works, one that you stick with over the long-term because it works for you and your life, it needs to be easy to add information to it. Really easy. Because if it’s not easy, you won’t do it. Neither will I. Friction kills productivity.

Throughout the day, there’s all kinds of information we should, need to, or want to save. Dates, reminders, random ideas, shopping lists, etc. To assist you, your PKM or productivity system needs these updates in as close to real-time as possible, so it stays in sync with your life.

Often, we come across web content — the perfect Reddit post, a great tutorial, a must try recipe — that we want to save for later. We may want to make sure we add it to our “read it later” reading list. Or catalog it into our recipe archive so we can put it into rotation for family meals.

If we bookmark it, it might disappear and we have yet another place to check for information we need contributing to that frustrating “I know it’s somewhere…” phenomena that often ends with Googling for it again, defeating the purpose of bookmarking or having a PKM solution.

Now, saving web content is easy.

Introducing the Ally web clipper Safari extension

Strip advertisements & clutter

The Ally web slipper automatically removes all the ads and complex formatting that plague modern websites so when you read it later, you can focus on the content. Read in Ally’s Full Screen mode for ultimate focus and immersion.

Permanent save & accessible files

With a single click, you can archive a permanent copy of whatever web page you’re on as a Markdown file. If the website disappears, if the article is put behind a paywall, or if you don’t have access to the internet, you’ll still have an archived copy of the web content you need.

Add notes for context

Add notes before saving the page so you’ll understand why you thought it was important later. Your future self will thank you. You can also add tags to automatically place it in your Library.

Custom keyboard shortcuts

Configure whatever keyboard shortcut you want for the fastest save possible. No reason to touch a mouse.

Automatic processing

Your web clippings and web content are automatically processed into vector embeddings and relationships between concepts and data are added to knowledge graph so you can be sure that the content you saved is easily surfaced when you need it. This also means that every web page you archive adds to Ally’s knowledge, so it’s more capable of assisting you with whatever you’re working on.

One click and it is added to your Inbox. That means you have a permanent copy that’s yours forever. If Reddit removes the post, if the website goes down, etc the data is now yours. The data is immediately chunked and vectorized. Each vector is analyzed for relationships to pre-existing data to form a knowledge graph of interrelated information.

This means with a single click you’ll store a copy of that information in markdown, turn it into searchable vector embedding, and add it to an evolving knowledge graph so your Ally can retrieve that information when it’s useful and relevant.

How to install the Safari browser web clipper extension on macOS

The Ally Web Clipper browser extension for macOS and iOS Safari is automatically included when you download the apps from the App Store. You don’t need to download anything additional after you’ve downloaded the apps.

  1. Download and install Ally
  2. Open Safari
  3. Go to Settings > Extensions
  4. Activate the Ally Web Clipper extension by clicking the box next to it

Using the web clipper

There are a two ways you can use the web clipper extension to save content:

Clicking the Ally icon in the toolbar

  1. Click the icon that appears next to the Safari address bar.
  2. A dropdown will appear that gives you the option to add any notes. This is a good idea if you often find yourself wondering why you saved something when you find it later.
  3. Click Save to Ally and it will automatically appear in your Unified Inbox

With configurable keyboard shortcuts

The quickest way to save web content is with a keyboard shortcut that bypasses the “add notes” step. When you use the keyboard shortcut, the web page is instantly saved and archived.

See the help doc on the Ally Web Clipper for more detailed information on setting up and using the web clipper browser extension to save web pages.
