Public beta: Advanced local LLM memory for Ally

If you go into Settings > AI and scroll below the list of LLMs, you’ll see a couple of new options to enable Ally’s advanced memory features:

Ally is designed to learn about you over time so that it can be more helpful. Since all of this happens completely locally and privately and no data is ever shared with anyone, ever, you don’t have to worry about some creepy company using what it learns to show you ads or leaking your data, etc.

A couple of notes:

  • Turning on memory features will slow response times. It takes additional time to search memories and it also takes additional time to load the additional context.
